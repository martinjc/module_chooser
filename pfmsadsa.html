Module Checker, MSc Advanced Computer Science (Full Time)

Selection Rules:

Students should select either:

  • CMT302 or CMT306, two optional autumn semester and two optional spring semester modules

or:

  • Three optional autumn semester modules and two spring semester modules. In this case, student will need to submit an 'unbalanced semester load waiver form' which must be signed by programme leader

or:

  • Two optional autumn semester modules and three spring semester modules. In this case, student will need to submit an 'unbalanced semester load waiver form' which must be signed by programme leader

In all cases, modules should be selected along research themes, so:

  • CMT106 and CMT202 should be selected together
  • CMT209 and CMT108 should be selected together
  • CMT107 and CMT206 should be selected together

No more than 2 security modules should be selected (CMT105, CMT104, CMT213, CMT306)
